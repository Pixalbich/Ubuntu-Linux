ls -позволяет показать список файлов в директории. 
pwd -позволяет в какой директории мы находимся.
cd - позволяет поменять директорию, "cd .." - вернуться в прошлую директорию
mkdir - для создания директорий.
cp - копировать.
rm - удалить, rm -rf - не юзать
mv - переместить или переименовать
touch - быстро создать пустой файл в текущей директории.
cat - много вариантов: cat > testfile - создаст новый файл и будет записывать текст из консоли в него. Закончить ввод Ctrl+D.
cat test test2 > test3 - склеить содержимое нескольких файлов в новый файл (test + test2 = test3). cat test3.txt - просмотр содержимого


Используя команду cat, создать два файла с данными, а затем объединить их. Просмотреть содержимое созданного файла. Переименовать файл, дав ему новое имя.
mkdir Test - создал папку
cd Test - перешел в папку
cat > test1txt - создали 1 файл
cat > test2.txt - создали 2 файл
cat test.txt test2.txt > test3.txt - склеиваем оба файла.
cat test3.txt - просмотр содержимого
mv test3.txt tekst.txt - переименовать

Создать несколько файлов. Создать директорию, переместить файл туда. Удалить все созданные в этом и предыдущем задании директории и файлы.
touch 1 - создаем 1 файл
touch 2 - создаем 2 файл
mkdir delete - создаем директроию
mv 1 delete - перемещаем в каталог delete
mv 2 delete - перемещаем в каталог delete
cd delete - переходим в директорию delete
ls - смотрим содержимое директории
cd .. - возвращаемся на шаг назад
rm -r Test - удаляем директорию Test с содержимым
rm -r delete - удаляем директорию delete с содержимым

Создать файл file1 и наполнить его произвольным содержимым. Скопировать его в file2. Создать символическую ссылку file3 на file1. Создать жесткую ссылку file4 на file1. 
Дать созданным файлам другие, произвольные имена. Создать новую символическую ссылку. Переместить ссылки в другую директорию.
mkdir test - папка где будем работать
cd test/ - переходим в созданную папку
touch file1 - делаем файл(пустой)
nano file1 - открываем в редакторе Нано (контрол + Щ - записывем и меняем имя на file2) (или ln file1 file2)
ln -s file1 file3 - символическая ссылка
ln file1 file4 - абсолютная ссылка
